How are the functional units connected? (Buses)

Coding is Love

For a computer to achieve its operation, the functional units need to communicate with each other. In order to communicate, they need to be connected.
Functional units may be connected by a group of parallel wires - called a bus. Each wire in a bus can transfer one bit of information. The number of parallel wires in a bus is equal to the word length of a computer

Information in a computer Instructions

Information fed to/stored in a computer is either 1- Instructions 2- Data Instructions specify commands to: Transfer information within a computer (e.g., from memory to ALU) Transfer of information between the computer and I/O devices (e.g., from keyboard to computer, or computer to printer) Perform arithmetic and logic operations (e.g., Add two numbers, Perform a logical AND).

Information in a computer Instructions

A sequence of instructions to perform a task is called a program, which is stored in the memory. Processor fetches instructions that make up a program from the memory and performs the operations stated in those instructions.

Input unit

Binary information must be presented to a computer in a specific format. The input unit roles: - Interfaces with input devices. - Accepts binary information from the input devices. - Presents this binary information in a format expected by the computer. - Transfers this information to the memory or processor.

Output unit

Computers represent information in a specific binary form. Output units Roles: - Interface with output devices. - Accept processed results provided by the computer in specific binary form. - Convert the information in binary form to a form understood by an output device.

Memory unit

Memory unit stores instructions and data. Data is represented as a series of bits. To store data, memory unit means storing bits.
Processor reads instructions and reads/writes data from/to the memory during the execution of a program.
Memory and processor have to “communicate” with each other in order to read/write information. In theory, instructions and data could be fetched one bit at a time.
In practice, a group of bits is fetched at a time.

Memory unit

Group of bits stored or retrieved at a time is termed as “word” Number of bits in a word is termed as the “word length” of a computer.

RAM provides fixed access time independent of the location of a word, known as "Memory Access Time".

Modern computers have three to four levels of RAM units with different speeds and sizes to reduce communication time.
